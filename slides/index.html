Architecture & Fonctionnalités

Stack : Rust + Bevy 0.11 — machine d'états, ECS, data layer JSON

🎮 Gameplay

  • Pilotage à la souris, physique de vaisseau
  • Ennemis IA (alien UFOs) avec IA de combat
  • Détection de collisions & mort
  • Score, timer, kill count, leaderboard

🌍 Scènes JSON

  • space_asteroids — ceinture de Saturne
  • ice_caves — cavernes d'astéroïde
  • desert_planet — dunes & montagnes
  • idf_transport — réseau RER/Métro 3D

🛸 Skins & Data Layer

  • Skins définis en JSON (formes, couleurs, émissifs)
  • Prévisualisations SVG rastérisées au démarrage
  • Carousel de sélection au menu principal

⚙️ Bevy ECS

  • États : StartMenu → Playing → Dead
  • Caméra 1ʳᵉ/3ᵉ personne (F5), spring-arm
  • HUD, minimap radar, keybindings JSON

🔗 Aspects TAL

  • LLM comprend la demande en langage naturel
  • Génère du JSON structuré valide
  • System prompt orienté game-design
  • Configurable via llm_config.json

🛠️ Détails techniques

  • HTTP via ureq dans un thread dédié
  • Résultat partagé par Arc<Mutex>
  • Jeu mis en pause pendant le chat
  • Clé API persistée dans secrets.json

Scène phare : IDF Transport

Le réseau RER & Métro parisien reconstitué en espace 3D — données réelles

🗺️ Réseau

  • 170+ stations géocodées
  • 19 lignes (RER + Métro)
  • Rails 3D élevés générés
  • Boundary : 500 000 unités

🚆 Trains IA

  • Patrouille de station en station
  • RER : 1 800 u/s
  • Métro : 1 200 u/s
  • Composant IdfTrain

📡 API Temps réel

  • PRIM API (IDFM)
  • Prochains départs live
  • Polling 30 s async
  • Mode démo sans clé
📍HUD proximité : nom de station + prochains trains à < 2 500 u
🎯Station Picker au menu : sélection des hubs à tracker
🔑Clé PRIM_API_KEY via env var ou secrets.json
⚡Requêtes dans un Arc<Mutex> partagé — jamais de blocage

Bilan & Perspectives

Ce qui fonctionne, ce qu'on a appris, et la suite

✅ Acquis

  • Prototype jouable bout en bout
  • Contenu totalement data-driven (JSON)
  • LLM génère & injecte du contenu live
  • Scène IDF avec vraies données PRIM

⚠️ Difficultés

  • Bevy 0.11 — tuple limit & !Sync types
  • LLM pas toujours fidèle au schéma JSON
  • SVG rasterisation lente au démarrage
  • Build long (> 5 min à froid)

🚀 Perspectives

  • GPU instancing & frustum culling
  • Validation JSON côté LLM (JSON Schema)
  • Effets sonores & particules
  • Multijoueur local (split-screen)

🎓 Enseignements TAL

  • LLM = générateur de contenu structuré
  • Prompt engineering crucial pour la fiabilité
  • Intégration non-bloquante indispensable en jeu
  • NLP + jeu vidéo = combo créatif puissant
